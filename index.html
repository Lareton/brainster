Анализ заданий

Для того чтобы делать релевантные рекомендации платформа должна понимать специфику задачи:
  • теоретические темы
  • сложность для декомпозиции
  • соотношение условия — решение (недостаточность/избыточность для решения)
    • Для этого мы выделяем явные и неявные признаки (эмбеддинги)

    стек используемых технологий:
  • clustering
  • tf-idf metric
  • nlp models (e.g. Bert)

    • Рекомендация заданий

    Спецификой задачи рекомендации заданий для решения является то, что мы не можем использовать неперсональные рекомендации - они будут просто бесполезны.

    применяемые технологии рекомендательных систем:
  • content based
  • collaborative filtering
  • graph based
